##Drag-Sort

本仓库使用远生JS实现了对下面代码中的block进行拖拽排序的功能。
```
<div class="drag-wrap">
	<div class="drag-block">1</div>
	<div class="drag-block">2</div>
	<div class="drag-block">3</div>
	<div class="drag-block">4</div>
	<div class="drag-block">5</div>
	<div class="drag-block">6</div>
</div>
```
